# Commercial Model

Mia uses:

- `AGPL-3.0-only` for the public community edition;
- a separate commercial license for proprietary deployment, embedding, and support;
- a CLA so maintainers can manage both models consistently.

## Why this model exists

The project includes significant product, security, and hosted-service logic. The AGPL keeps the public core open, while the commercial path supports proprietary deployments and funded maintenance.

## Community edition

The community edition is intended for:

- learning;
- self-hosting with AGPL compliance;
- public contributions;
- transparent core development.

## Commercial edition

The commercial path is intended for:

- proprietary use;
- internal enterprise deployments with non-public modifications;
- commercial warranties, support, and contractual terms.
