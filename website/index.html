System

Real agent infrastructure, not a chatbot wrapper.

01

iMessage interface

SendBlue webhooks receive inbound messages, dedupe events, and send replies back through iMessage.

02

Parent router

LangGraph classifies intent and creates dynamic sub-agent specs with isolated tool access.

03

Memory system

Convex stores short-term, long-term, and permanent memories with importance and decay metadata.

04

Memory Court

A nightly workflow consolidates, challenges, and judges memory cleanup decisions with audit records.

05

Gateway

A single local process starts the dashboard, Convex dev, and the Python agent service.

06

Dashboard

A realtime Next.js interface shows agents, memory, events, automations, and setup state.

Stack

Get started

Run Mia locally in three steps.

1. Clone the repo and install dependencies.

git clone https://github.com/zhiliao000-star/mia-agent.git
cd mia-agent
npm install

2. Copy .env.example to .env.local and fill in your keys.

cp .env.example .env.local

3. Start all services with the Mia Gateway.

npm run mia:gateway
