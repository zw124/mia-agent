# Roadmap

## Near term

- stabilize the desktop release path;
- harden onboarding and deployment configuration;
- improve tool trace visibility and memory/session continuity;
- formalize contributor workflow and security review gates.

## Mid term

- production-grade auto-update flow with signed desktop artifacts;
- clearer plugin and integration boundaries;
- hosted account and deployment model refinement;
- release automation and compatibility policy.

## Long term

- stronger multi-agent orchestration;
- safer remote execution and approval policies;
- public extension ecosystem with explicit API contracts;
- commercial support and enterprise deployment packaging.
